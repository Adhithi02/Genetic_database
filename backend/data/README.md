# Data Directory

This directory contains all raw biological datasets and processed outputs used by the genomic analysis pipeline.

## Files

### Raw Datasets (Input)
- **Ensembl2Reactome_All_Levels.txt**: The complete Reactome pathway database mapping Ensembl gene IDs to biological pathways across all organisms. Source: [reactome.org](https://reactome.org/download-data)
- **brenda_2026_1.txt**: The complete BRENDA enzyme database in Swissprot/Trembl text format, containing enzyme kinetic parameters (Km, turnover numbers) for all catalogued organisms. Source: [brenda-enzymes.org](https://www.brenda-enzymes.org/download.php)

### Processed Output
- **biology_reference.json**: The compiled biological dictionary generated by `scripts/process_biology_data.py`. Contains pathway mappings and enzyme kinetics for 3,464 human genes extracted from the GWAS dataset. This file is loaded into memory by FastAPI at startup for instant biological interpretation of predictions.

## Notes
- The raw datasets are large (250MB+ each) and should be listed in `.gitignore` if not already.
- `biology_reference.json` is the only file required at runtime. The raw datasets are only needed if you want to regenerate it.
